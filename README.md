# 🧮 Calculator Web App

A simple, modern, and responsive calculator built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations and includes advanced features like dark mode, keyboard support, and calculation history.

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 🌙 Dark / Light Theme Toggle
- ⌨ Keyboard Support
- 🕒 Calculation History
- 📱 Fully Responsive Design
- 💾 Local Storage Support

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
